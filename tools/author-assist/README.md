# Author Assist tool

Static mock tool for testing content prompts without any external API calls.

## How to open on preview
- Go to your preview environment at `/tools/author-assist/`, e.g. `https://feature-eds-author-assist-tool--edge-space--andy-lambert.aem.page/tools/author-assist/` (or swap the hostname for another branch/main).
- Fill in the Preview URL, pick an intent and tone, add optional constraints, then click **Run**. A mocked response with suggested edits appears in the output panel.
